For better scrolling experience, you may want to disable text selection/highlighting and image dragging. Here's how you can do it (if you need to).

Prevent Image Dragging 
<img src="someImage.png" draggable="false" onmousedown="if (event.preventDefault) event.preventDefault()">
Prevent text highlighting 

CSS                 
.no-text-select{
    -webkit-touch-callout: none; /* iOS Safari */
    -webkit-user-select: none; /* Safari */
    -khtml-user-select: none; /* Konqueror HTML */
    -moz-user-select: none; /* Firefox */
    -ms-user-select: none; /* Internet Explorer/Edge */
        user-select: none; /* Non-prefixed version, currently
                                    supported by Chrome and Opera */
}

HTML
<p class="no-text-select">
    Unselectable text.
</p>
